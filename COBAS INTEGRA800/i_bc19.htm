Block 19: Order manipulation response

Each request with block code 10, 11 or 12 is answered by block 19, the order manipulation response. The block's content depends on the request.

The response message structure depends on the success of the request and also on the request type. One of the following block structures could appear:

Response to an order entry / test addition:
Response to a correct order entry / addition:
96 Error Code
Response to an order with a line format error:
96 Error Code
Response to an incorrect order
entry / addition with error in patient:
96 Error Code
50 Patient ID
Response to an order entry/addition with error in sample order:
96 Error Code
53 Order ID
54 Order Information 0 to 1
Response to an incorrect order entry with error in test order:
96 Error Code 1 to x
55 Test ID
Response to a test addition with default sample type, if the previously used sample type was not a default sample type:
96 Error Code 1 to x
53 Order ID
55 Test ID
Response to an incorrect special order:
96 Error Code
52 Specific Order Selection
55 Test ID
x = 1 to 50 for Integra 700
x = 1 to 200 for Integra 400

 

Response to an order deletion / test deletion:
Response to a correct order/test deletion:
96 Error Code
Response to an order / test deletion with error in line format:
96 Error Code
Response to an incorrect order / test deletion with fault in sample order
96 Error Code
53 Order ID
Response to an incorrect test deletion:
96 Error Code 1 to x
55 Test ID
Response to an order/test deletion with default sample type if the order’s sample type was not default sample type:
96 Error Code 1 to x
53 Order ID
55 Test ID
x = 1 to 50 for Integra 700
x = 1 to 200 for Integra 400

 

Response to an order priority modification: (Integra 700 only)
Response to a correct priority modification:
96 Error Code
Response to incorrect priority modification with fault in action
96 Error Code
57 Priority Modification
