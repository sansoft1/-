Error Codes

COBAS Integra uses the following error codes:

  • Line code 96 error codes
  • Line code 99 error codes
  • System error messages (Integra 700 only)

700  Host Interface System Error Messages

System error messages are shown on the Data Station's user interface and are added to the event.log file.

HIF-001 General HIF Error

HIF-002

Line Error: Received line could not be processed, e.g. 9414 Host interface enabled but 9561 data received

HIF-003

Block Error: Received block could not be processed

HIF-005

Port Error: Serial interface port problem detected
HIF-008 Order Error
HIF-009 Patient Error
HIF-010 Result Error
HIF-011 DB Status Error
HIF-013 System Error
HIF-014 Sample Type Error: Undefined sample type on barcode
