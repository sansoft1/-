Line Code 10: Result type selection

 

Description

Format

Range

Line Code

10

    

Result Type Selector

I2
01 = Transmit any available result, i.e.
- single patient or
- single calibration or
- control result collated by Test ID
02 = Reserved
03 = Transmit patient results, collated by Order ID.
Results are ONLY transmitted if ALL tests belonging to an order are finished and accepted.
(This "Result type selector" is available starting with software version  9561.B1.)

Unless really required, do not use "Result Type Selector" 03, 04,
05 or 06
(refer to the following note).
04 = Transmit patient results, collated by Order ID.
Blocked tests are marked with the ‘Test Blocked’ flag (CALC flag 255). Their results are sent later as soon as successfully completed along with any still-remaining blocked tests (9561.B1).
05 = Transmit patient results, collated by Order ID.
Send all currently available results (i.e., all accepted but not yet transmitted results) (9662.A1)
06 = Transmit new control results collated by Control ID (9662.A1)
07 = Transmit single patient result (9561.B3)
08 = Transmit single calibration result (9561.B3)
09 = Transmit control results collated by Test ID (9561.B3)

Example: 10_07<LF>

Note: - Only accepted and not yet transmitted results are sent to the Host.
. - Due to instrument performance reasons and in order to have results on the Host system available as soon as possible,  we strongly recommend NOT using "Collated Results" requests ( i.e., not using Result Type Selector 03, 04 and 05 )
. - Calibration and Control results are only transmitted if enabled. Integra setup for:  700 / 800 / 400
