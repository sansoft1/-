DATA FORMATS

Format-Code

Format-Description

Sx

x-character ASCII-String, left adjusted,
Length x is mandatory, i.e. a shorter string has to be increased with <space>s to the length x. However the trailing spaces are not stored.
Example: S15 "Mr. Evans······"

Ix

ASCII-formatted x-digit integer value, right adjusted,
Length x is mandatory, shorter integer-value has to be increased with <space> or 0 on the left to the length x.
Example: I5 "00069" or "···69"

SIx

ASCII-formatted x-digit integer value with sign, right adjusted,
Length x is mandatory, shorter integer-value has to be increased with <space> or 0 on the left to the length x.
Example: SI5 "-0069" or "-··69" or "+··78" or "·0078"

Rx.y

ASCII-formatted real-value, right adjusted
x = Length of the whole data record
y = Number of digits after the decimal point (optional)
Length x is mandatory, shorter real-value has to be increased with <space> or 0 on the left to the length x.
The decimal point is also mandatory.
Example: R8.3: "0077.123", Example: R6: "·8.111" or "00045."

SRx.y

ASCII-formatted real value with sign, right adjusted
x = Length of the whole data record
y = Number of digits after the decimal point (optional)
Length x is mandatory, shorter real-value has to be increased with <space> or 0 on the left to the length x.
The decimal point is also mandatory.
Example: SR9.3: "-0077.123",Example: SR7: "····8.1" or "+00045."

Fx

ASCII formatted floating point number, right adjusted
x = Length of the whole data record
Length x is mandatory, shorter real-value has to be increased with <space> or 0 on the left to the length x:
The sign "+" or "-" for exponent is mandatory
Decimal point and exponent "E" are mandatory

Mantissa has exactly one digit before the decimal point in the range 1..9
Between mantissa and exponent no <space> is allowed
Example: F12: "····6.11E+33", Example: F15: "-00007.4321E-02"

Hx

x-digit hexadecimal value
Example: H2 "1C"

DT1

Date and Time format: DD.MM.YYYY hh:mm:ss (hh in 24 h format!)
19 character ASCII-string
Example: "12.02.1998 22:07:34"
