Enable Host Interface Transmission

The COBAS Amplicor's interface requires no extra enabling. The Host interface is available as soon as the message `System Init' is printed and/or the `Status Display' is shown.

Additional Parameter

Reset Host Interface

No / Yes

Due to the fact that the COBAS Amplicor provides `Power Fail' features, it resumes possibly interrupted transmissions after a restart. Therefore the `Reset Host Interface' feature may be needed if a Host system is not able to cope with this feature. The Host interface is reset by activating the parameter, leaving the menu and switching the Instrument Off - On !

_ Specific result requests are supported.

Programmer's Guideline

The general communication rules are explained in section `2. Basics'. This chapter illustrates the instrument specific details.

Order Entry

· Orders are downloaded by means of block 10.

· An Order Entry Block may contain all orders to one A-ring with up to 72 qualitative tests or with up to 24 quantitative tests attached to it.

· Orders can be downloaded as soon as the information becomes available on the Host.

Add Tests

Adding tests to orders which exist already on the COBAS Amplicor is done by means of an Order Entry block (Block 10). When adding tests, the already known information (order and tube information lines) is omitted. Please refer to Block 10 description `Add Tests to existing A-ring/A-tube'.

Rerun Orders

Unlike the user interface the Host interface does not support order reruns. Reruns are performed by adding the required test again to the existing order. Please refer to Block 10 description `Add Tests to existing A-tube'.

Result Request

Note the following important details:

· Due to the Master-Slave principle, results are transmitted to the Host only following the receipt of a Result Request block (Block 09).

· COBAS Amplicor accepts specific and non-specific result requests.

· When using non-specific result requests, each result will be transmitted only once. If results have to be re-transmitted, this can be done by using specific result requests.

· Results on the COBAS Amplicor are provided within a cycle time of 72 seconds. It therefore takes at least this period until a next result will be available.

Time Out

An exactly defined response time cannot be guaranteed. However a response should be returned within 25 seconds plus an additional transmission time. We therefore suggest to use a time out of at least 30 seconds.
