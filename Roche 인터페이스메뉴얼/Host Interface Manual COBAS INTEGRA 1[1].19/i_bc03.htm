Block 03: Control Result

Block 03 is transmitted to the Host after receiving a result request 
- and if an accepted and not yet transmitted control result is available
- and if the instrument is configured accordingly. Integra 700 / 800 / 400

55 Test ID 1 to m
01 Result Time
02 Control ID 1 to n
00 Result Data

 

Results collated by Test ID ( Result type selector ’01’ or ‘09’ is used ):
m = 1 ( one Test ID per result block )
n = 1 to 3 : for Integra 700 / 800 ( n is the number of control sera which is defined in test parameters )
n = 1 to 6 : for Integra 400
Note:  Results of different Control Orders are not collated  ( Integra 400 only ).
.
Results collated by Control ID ( Result type selector ’06’ is used ):
m = 1 to # of available tests ( m is the number of tests with not yet transmitted QC results )
n = 1 ( one Control ID per result block )
