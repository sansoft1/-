Block 61: Slot configuration

The reply to an MCS request with service = 0 is Block 61 with the current slot configuration. 

41 Slot State

The rack numbers of the racks in the slots are sent.

 

For INTEGRA 800, however, line 41 is repeated five times:

41 Slot State ( Slot 1 to 5 )
41 Slot State ( Slot 6 to 10 )
41 Slot State ( Slot 11 to 15 )
41 Slot State ( Slot 16 to 20 )
41 Slot State ( Slot 21 to 25 )

 

INTEGRA 800:   Definition of slots
Slot 1 - 16: 16 carousel positions for RD 5 racks
Slot 17 - 18: 2 positions for RD 15 racks
Slot 19 - 24: 6 positions for Inserts on Multi rack ( occupies the two pos. of slots 17 and 18 )
Slot 25: ISE rack
