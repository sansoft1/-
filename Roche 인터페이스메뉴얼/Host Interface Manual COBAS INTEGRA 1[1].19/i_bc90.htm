Block 90: System status request ( INTEGRA 800 only )

Block 90 is used to interrogate the system status of the INTEGRA 800. Block 90 has no lines.

 

Block 91: System status response ( INTEGRA 800 only )

Block 91 is the response to Block 90. It contains line 97.
