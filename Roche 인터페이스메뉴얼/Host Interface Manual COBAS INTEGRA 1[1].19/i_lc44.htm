Line Code 44: Cal / CS state

 

Description

 Format Range

Line Code

 44  

CAL/CS state type

 S1 C = Calibration state
O = Correction state
Q = Control state

CAL/CS state

 S1 A = Accepted
C = Calculated
P = Pending
R = Reactivated
F = Failed (Calibrations and Corrections only)

Time of last Cal/CS issue

T1     

Date of last Cal/CS issue

D1 01/01/1800 to 31/12/2100

Example: 44_C_A_10:32:45_30/11/1993<LF>
