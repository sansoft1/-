Rack Number:  (Integra 800 only)

There are two different types of rack numbers used with the INTEGRA 800:

  1. An unambiguous Rack Number for all Racks (barcoded), which is used internally and on the Host Interface.
  2. A "visual ID" used on the User Interface.
