Line Code 57: Priority Modification

 

Description

Format

Range

Line Code

57

     

Priority modification type

I2

 1 = change work list priority to Active (only Integra 700)
2 = change work list priority to STAT

Example: 57_•1<LF>

Note:
It is only possible to change from Passive (P) to Active(A), from Passive (P) to STAT, or from Active (A) to STAT.
