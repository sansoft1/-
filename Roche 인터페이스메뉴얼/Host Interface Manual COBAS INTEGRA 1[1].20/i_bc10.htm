Block 10: Order entry

There are different possibilities for entering orders into the COBAS INTEGRA work list.
The database set-up ( Integra 700 / 800 / 400 ) defines whether the Instrument runs in patient mode or sample mode.
No patient information is required if in sample mode.

New sample order:
50 Patient ID 0 to w
53 Order ID
54 Order Information
55 Test ID 1 to x
Add tests to an existing
sample order:
50 Patient ID 0 to w
53 Order ID
55 Test ID 1 to x
w = 1 if in patient mode
w = 0 if in sample mode
x = 1 to 50 for Integra 700 / 800
x = 1 to 200 for Integra 400
Warning: 
( applies only if in Patient mode )
     Patient ID has to be unique. 
>> Do not use the same Patient ID for more than one patient, otherwise results could be assigned to the wrong patient !!
