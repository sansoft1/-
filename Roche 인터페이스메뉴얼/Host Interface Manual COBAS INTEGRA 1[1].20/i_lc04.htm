Example: 04_•1.000000E+00_-1.513617E-02_-5.716211E-04<LF>

The meaning of parameters 1 to 5 depends on the calibration mode used.

- For qualitative calibration, parameter 1 corresponds to the cut-off value.
- For quantitative calibration, parameters 1 to 5 correspond to the curve parameters of the model used.
- For ISE calibration, the measured electrode defines the parameter. If the Li electrode has been measured, parameter 1 contains the Li slope, parameter 2 the Na slope and parameter 3 the K slope.
- For all other electrodes, parameter 1 contains the associated slope value.
