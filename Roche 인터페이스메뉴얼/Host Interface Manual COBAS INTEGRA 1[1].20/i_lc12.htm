Line Code 12: Control lot Information

 

Description

Format

Range

Line Code

11

     

Reagent Lot

S10

Reagent Expiration Date

D1

 dd/mm/yyyy

Example: 11_N123A•••••_23/04/2006<LF>

Notes: - If no lot information is found or no Lot was defined, "??????????" will be used and transmitted (without quotes). 
- If no expiration date is given for the lot, "00/00/0000" will be used and transmitted (without quotes). 
- If the Lot information is shorter than ten characters, the string will be filled with spaces on the right side.
  Example, N123A will be   N123A • • • • •            (•) represents a filler space
