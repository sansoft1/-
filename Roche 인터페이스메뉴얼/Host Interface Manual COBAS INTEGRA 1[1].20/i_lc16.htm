Notes regarding Result Indexes:
- A calibration result comprises only the Calibration Index with which the calibration itself will be identified.
- A control result is comprised of the Calibration Index ( reference to the calibration with which the control was calculated) and
  a control group index, identifying the control itself.
- A Sample result is comprised of
  • A calibration index ( Reference to the calibration with which the sample result was calculated) 
  • If there is an associated control, then there is a Control Group index (reference to the control).
  • A sample index, with which the result itself will be identified
- A result will have a maximum of three indexes. If one index does not exist, it will be replaced with a right-justified zero. See the
  examples above.
