Host Interface new features:
New features in the cobas HIF protocol are introduced with the Integra 400 version 3.4 and the
Integra 800 software version 9864.b1.

The protocol is backwards compatible, this means that  is has been extended, and not  just modified.

The new features essentially introduce new Result type selectors and the resulting response blocks from the cobas INTEGRA instruments.

THe new blocks feature new line codes and are intended to enhance the traceability of results, by sending additional information:

  • Date/time stamp
  • Calibration lots
  • Reagent lots

The communication protocol is similar for all COBAS Instruments.
New or modified codes:
block codes New Block codes 05, 06, 07 ; Block code 09 has extended functions (see line code 10 below)
line codes New line codes 11, 12, 13, 15, 16. Line code 10 has been extended with result type selectors 11, 17, 18, 19.
