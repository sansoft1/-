Block 05: Calibration Result with lot information

Block 05 is transmitted to the Host after receiving a result request with result type selector ‘11’ or ‘18’
- and if an accepted and not yet transmitted calibration result is available
- and if the instrument is configured accordingly. Integra 700 / 800 / 400

55 Test ID
15 Access Date Time
16 CALidx
03 Standard Rates 1 to y+1
04 Calibrate Curve
13 Calibrator Lot Information
11 Reagent Lot Information
00 Result Data 1 to m
07 ABS Sample Check 0 to y

 

m = 1 to 6 m is the number of standard rates in line 03. The sequence of m iterations must correspond to the sequence of values in line 03. In addition, the number of iterations m has to be equal to the number of parameters (standard rates) in line 03.
y = 0 or 1 The value of y depends on whether the antigen excess (AGE) check is switched ON or OFF. The COBAS INTEGRA always transmits an AGE rate line regardless of this parameter. If the AGE check was not required by the test, this line contains the values initialized by the instrument.
