Block 06: Control Result with lot information

Block 06 is transmitted to the Host after receiving a result request  with result type selector ‘11’ or ‘19’
- and if an accepted and not yet transmitted control result is available
- and if the instrument is configured accordingly. Integra 700 / 800 / 400

55 Test ID 1 to m
15 Access Date Time
16 CALidx
11 Reagent Lot Information
02 Control ID 1 to n
12 Control Lot Information
00 Result Data

 

Results always collated by Test ID ( Result type selector 11’ or ‘19’ is used ):
m = 1 ( one Test ID per result block )
n = 1 to 3 : for Integra 700 / 800 ( n is the number of control sera which is defined in test parameters )
n = 1 to 6 : for Integra 400
Note:  Results of different Control Orders are not collated  ( Integra 400 only ).
