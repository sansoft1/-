Block 63: Cal / CS state (Integra 700 only)

An MCS request with service = 2 returns a Block 63.

43 Test State 1 to x
44 Cal/CS State 3

x = 1 to max. number of active tests.

Block 63 contains the relevant calibration and control status of all tests defined in the COBAS INTEGRA 700. For each test a line 43 and three lines 44 (a calibration, a correction and a control state) will be transferred.
